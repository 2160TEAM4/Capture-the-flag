Introduction

We will have a simple Capture the Flag (CTF) activity for you to complete. You will have a terminal line where you have to find the flag in Linux machine.

The flag will be a file in the Linux file system that you must find and allow to be read. Once that file has been read, the CTF will be considered complete. A cheat sheet and instructions file will be provided.

Step 1

Make sure you have these tools on your computer, Here's how to get started:

  • Terminal emulator (such as terminal on Mac or command prompt on Windows)
  • Python 3.x
  • Git

If you find that your computer does not have these tools, please call the teacher.

Step 2

Game challenges and tokens are stored in Git repositories. First, clone the repository to the local machine using the following command:

git clone https://github.com/2160TEAM4/Capture-the-flag.git

Replace [repository URL] with the URL of the repository provided by the game administrator. Once you've cloned the repository, you're ready to play.

Step 3

This flag will be a file in the Linux file system that you must find, and grant read permission to. Once the file has been read, the CTF will be considered complete. A cheat sheet and instruction sheet will be provided.

Are you ready? Join the game and feel the charm of computer science!

Attention! You only have 11 minutes.
